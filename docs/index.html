Installation

Get started with V Web in your project:

npm install v-web

Or using yarn:

yarn add v-web

Quick Start

Create your first V Web application:


import { VWeb } from 'v-web';

const app = new VWeb({
    element: '#app',
    data: {
        message: 'Hello V Web!'
    }
});
